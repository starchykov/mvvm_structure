# mvvm_structure

Sample Folder Structure for applications with MVVM Architecture.

root application
└─── data
     └─── repositories
     └─── data providers
└─── domain
     └─── entities
     └─── constants
     └─── clients
     └─── services
└─── ui
     └─── view_state
     └─── view_model
     └─── presentation_widget
└─── utils
